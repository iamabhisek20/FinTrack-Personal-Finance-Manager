# FinTrack-Personal-Finance-Manager
FinTrack is a responsive personal finance manager built with HTML, CSS, and JavaScript. It helps users track income, expenses, savings goals, and cash flow using interactive dashboards and charts. Data is stored securely in browser localStorage with no backend required.
